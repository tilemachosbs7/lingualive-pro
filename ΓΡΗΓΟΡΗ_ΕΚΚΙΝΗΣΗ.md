# 🚀 Οδηγός Γρήγορης Εγκατάστασης

## Βήμα 1: Εγκατάσταση Extension

1. **Άνοιξε το Chrome** και πήγαινε στο: `chrome://extensions/`
2. **Ενεργοποίησε** το "Developer mode" (πάνω δεξιά)
3. **Κλικ στο** "Load unpacked"
4. **Διάλεξε** το φάκελο: `C:\Users\tilem\Desktop\projects\LanguageTranslate\extension`
5. ✅ Το extension είναι εγκατεστημένο!

---

## Βήμα 2: Εκκίνηση Backend

### Αυτόματη εκκίνηση (Εύκολο):
```powershell
cd C:\Users\tilem\Desktop\projects\LanguageTranslate
.\start-backend.ps1
```

### Χειροκίνητη εκκίνηση:
```powershell
cd C:\Users\tilem\Desktop\projects\LanguageTranslate
.venv\Scripts\Activate.ps1
cd backend
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

## Βήμα 3: Ρύθμιση API Keys

1. **Άνοιξε το αρχείο**: `backend\.env`
2. **Πρόσθεσε τα API keys σου**:
   ```env
   # Transcription Providers
   DEEPGRAM_API_KEY=4c0d35de8a786a374c81b982825bf3882b38dfa6
   GOOGLE_CLOUD_CREDENTIALS=./backend/google-credentials.json

   # Translation Providers
   DEEPL_API_KEY=8a8f0bbf-9c0b-4a5b-bc2d-0017b1d5bc5c:fx
   
   # OpenAI (για fallback)
   OPENAI_API_KEY=sk-proj-... (το δικό σου)
   ```

---

## Βήμα 4: Ξεκίνα να χρησιμοποιείς!

### Τρόπος 1: Μέσω Popup (Συνιστώμενο)
1. **Κλικ** στο εικονίδιο του extension (πάνω δεξιά στο Chrome)
2. **Διάλεξε** providers (Deepgram + DeepL συνιστώνται)
3. **Κλικ** στο **"▶️ Ενεργοποίηση HUD"**
4. Το HUD θα εμφανιστεί στη σελίδα!

### Τρόπος 2: Αυτόματο (Legacy)
- Το HUD βγάζει popup αυτόματα όταν ανοίγεις νέα σελίδα
- Κάνε κλικ το κουμπί για να το ανοίξεις

## 🆘 Troubleshooting

### Το popup δεν ανοίγει
- Έλεγξε αν το extension είναι enabled
- Κάνε refresh το extension
- Επανεκκίνησε το Chrome

### Δεν βγάζει transcription
- Βεβαιώσου ότι το backend τρέχει
- Έλεγξε τα API keys στο .env
- Δώσε άδεια για tab audio capture

### Κακή ποιότητα μετάφρασης
- Δοκίμασε DeepL αντί για OpenAI
- Έλεγξε ότι διάλεξες τη σωστή γλώσσα
- Βεβαιώσου ότι το audio είναι καθαρό
