# 🎯 Οδηγός Χρήσης - LanguageTranslate

## 🚀 Πώς να ξεκινήσεις

### 1. Άνοιγμα της εφαρμογής
- Κάνε κλικ στο εικονίδιο της επέκτασης στη γραμμή εργαλείων του Chrome
- Θα ανοίξει το **popup control panel**
- Πάτα το κουμπί **"▶️ Ενεργοποίηση HUD"**
- Το HUD θα εμφανιστεί στη σελίδα σου!

### 2. Έναρξη μετάφρασης
- Μέσα στο HUD, πάτα **"Start captions"**
- Δώσε άδεια στο Chrome να καταγράψει τον ήχο της καρτέλας
- Τα captions θα εμφανιστούν αυτόματα!

## 🎙️ Επιλογές Αναγνώρισης Φωνής (Transcription)

### ⚡ **Deepgram** (ΣΥΝΙΣΤΏΜΕΝΟ)
**Τι κερδίζεις:**
- ✅ **Ταχύτερο**: ~300ms latency
- ✅ **Πραγματικός real-time**: Βλέπεις κάθε λέξη καθώς λέγεται
- ✅ **Καλύτερη ποιότητα**: Word-by-word streaming
- ✅ **Δωρεάν**: $200 credit κατά την εγγραφή

**Πότε να το χρησιμοποιήσεις:**
- Για live streaming, meetings, webinars
- Όταν θέλεις άμεση απόκριση
- Για επαγγελματική χρήση

---

### 🔵 **Google Speech**
**Τι κερδίζεις:**
- ✅ **Μέτριο**: ~500ms latency
- ✅ **Δωρεάν tier**: 60 λεπτά/μήνα
- ✅ **Αξιόπιστο**: Τεχνολογία Google

**Πότε να το χρησιμοποιήσεις:**
- Όταν δεν έχεις Deepgram API key
- Για περιστασιακή χρήση (κάτω από 60 λεπτά/μήνα)
- Ως backup επιλογή

---

### 🤖 **OpenAI Realtime**
**Τι κερδίζεις:**
- ⚠️ **Πιο αργό**: 1-2 δευτερόλεπτα latency
- ✅ **Πάντα διαθέσιμο**: Δουλεύει με το υπάρχον OpenAI API key
- ✅ **Καλή ποιότητα**: Transcription + translation σε ένα

**Πότε να το χρησιμοποιήσεις:**
- Ως fallback όταν τα άλλα δεν δουλεύουν
- Όταν η ταχύτητα δεν είναι κρίσιμη
- Για ηχογραφημένο περιεχόμενο (όχι live)

---

## 🧠 Επιλογές Μετάφρασης (Translation)

### 💎 **DeepL** (ΣΥΝΙΣΤΏΜΕΝΟ)
**Τι κερδίζεις:**
- ✅ **Καλύτερη ποιότητα**: #1 στον κόσμο για ευρωπαϊκές γλώσσες
- ✅ **Φυσικότερες μεταφράσεις**: Κατανοεί το context καλύτερα
- ✅ **Ιδανικό για Ελληνικά**: Εξαιρετικά αποτελέσματα ΕΛ↔ΕΝ
- ✅ **Δωρεάν tier**: 500,000 χαρακτήρες/μήνα

**Πότε να το χρησιμοποιήσεις:**
- Για επαγγελματικό περιεχόμενο
- Όταν χρειάζεσαι υψηλή ποιότητα
- Για ευρωπαϊκές γλώσσες (ΕΛ, FR, DE, ES, IT κτλ)

---

### 🤖 **OpenAI**
**Τι κερδίζεις:**
- ✅ **Καλό για όλες τις γλώσσες**: Υποστηρίζει 100+ γλώσσες
- ✅ **Syntax checking**: Διορθώνει λάθη αυτόματα
- ✅ **Ήδη το έχεις**: Χρησιμοποιεί το υπάρχον API key

**Πότε να το χρησιμοποιήσεις:**
- Για exotic γλώσσες που δεν υποστηρίζει το DeepL
- Ως backup επιλογή
- Όταν θέλεις syntax correction

---

## 📊 Σύγκριση Συνδυασμών

### 🏆 **Καλύτερος Συνδυασμός** (Συνιστώμενο)
```
🎙️ Deepgram + 🧠 DeepL
```
- Ταχύτατο (300ms transcription)
- Άριστη ποιότητα μετάφρασης
- Ιδανικό για live streaming

---

### 💰 **Δωρεάν Συνδυασμός**
```
🎙️ Google Speech + 🧠 DeepL
```
- Καλό για <60 λεπτά/μήνα
- Δωρεάν και αξιόπιστο
- Καλή ποιότητα

---

### 🔧 **Fallback Συνδυασμός**
```
🎙️ OpenAI + 🧠 OpenAI
```
- Ένα API key για όλα
- Πιο αργό αλλά αξιόπιστο
- Καλό για testing

## 💡 Tips

- ✅ Ελαχιστοποίησε το θόρυβο φόντου
- ✅ Δοκίμασε διαφορετικούς providers για το use case σου

### Κόστος:
- **Deepgram**: $0.0043/λεπτό μετά το δωρεάν credit
- **Google Speech**: Δωρεάν για 60 λεπτά/μήνα
- **DeepL**: Δωρεάν για 500K χαρακτήρες/μήνα
- **OpenAI**: Σύμφωνα με το plan σου
